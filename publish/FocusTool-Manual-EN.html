FocusTool 3.2.1

Pointer effects, screen annotation, visual masks, timers, captures, boards, and a window presentation view in one Windows utility.

Get startedShortcut tableРусское руководство
Purpose. FocusTool adds visual presentation tools to other applications without modifying their source content.

1. Quick start

  1. Run FocusTool.exe. Its icon appears in the Windows notification area.
  2. Open the toolbar with Ctrl+Alt+G, or use the tray icon menu.
  3. For a brief pointer, hold Alt+Z (laser) or Alt+X (cursor highlight).
  4. Press Ctrl+Alt+A to draw. Pick a toolbar tool or press its key, such as W for Pencil or F for Highlighter.
  5. Press Esc to cancel the current operation or leave an interactive mode. If an editor is active, the first Esc closes that editor.

All shortcuts shown here are defaults. Change or disable them under Settings → Shortcuts. Existing customized settings from an earlier version are preserved.

Windows SmartScreen. Windows may warn about an unsigned build. Only run an archive obtained from a source you trust.

2. How the utility works

Global modes

Laser, highlights, timers, and other tools can be enabled over your current work without switching to a FocusTool window.

Interactive modes

Drawing, region selection, and boards accept mouse and keyboard input. Esc cancels a sub-mode or closes them.

Multiple monitors

Overlays use desktop coordinates across multiple displays. A full screenshot applies to the current monitor.

Above windows

The overlays are intended for normal desktop windows. Exclusive full-screen apps, protected video, and some system surfaces may bypass them.

Most modes are available from the toolbar, tray menu, or an assigned global shortcut. A Hold shortcut works only while held; a Toggle shortcut switches a persistent mode.

3. Pointer and click feedback

Laser Pointer

A colored point follows the cursor and leaves a fading trail. Color slots, point size, trail and fade duration, glow, and Always/Hold behavior are configurable. Toggle: Ctrl+Alt+Z; hold: Alt+Z.

Cursor Highlight

A stable ring around the cursor, with colors, radius, and thickness independent of the laser. Toggle: Ctrl+Alt+X; hold: Alt+X.

Click Pulse

A brief visual pulse at the click position. It is enabled independently of the cursor ring and has its own Ctrl+Alt+C shortcut. It is visual feedback, not an input logger.

Tip. For left-hand control, keep brief actions on Alt+Z/X and persistent modes on the corresponding Ctrl combinations.

4. Focus and magnification

Spotlight

Dims everything outside a circle around the cursor. Configure its radius and dim opacity. Toggle it with Ctrl+Alt+S, or hold Alt+S temporarily. While active, Ctrl+wheel changes dim opacity and Ctrl+Shift+wheel changes the radius.

Magnifier

A circular cursor-following magnifier with configurable radius and zoom. Toggle it with Ctrl+Alt+V. Use Ctrl+wheel for zoom and Ctrl+Shift+wheel for radius. It is briefly hidden during a regular capture so it does not obscure the chosen area.

Spotlight with Magnifier

Both modes can be enabled together. Content is magnified inside the clear area while the rest of the screen remains dimmed. The shared opening follows the Magnifier radius, creating a focused lens that works well for presenting small interface elements.

Pinned Lens

Create a separate pinned lens with Ctrl+Alt+F: first select a source region, then position the new window. Multiple lenses are supported. Move, resize, zoom, freeze, and close them. Ctrl+wheel over a lens changes its zoom. The context menu offers Freeze/Unfreeze, zoom in/out, Close, and Close all. Freeze is useful for comparing a fixed state with a changing screen.

Region Spotlight

Press Ctrl+Alt+Shift+S and draw a rectangle; the outside is dimmed. The regular circular Spotlight is turned off when region selection begins so it does not interfere. Create multiple regions, then move or resize them. Arrow keys move the selected region by 1 pixel and Shift+arrow by 10; Delete removes it and Enter finishes editing. Clear all regions from the toolbar or tray; no clear shortcut is assigned by default.

Compatibility. Lenses use the desktop image made available by Windows. Protected video and some hardware surfaces may not provide a frame for magnification.

Spotlight and Magnifier are hidden on Screen, Black, and White Board so the board remains a self-contained workspace.

5. Drawing and annotations

Toggle Draw with Ctrl+Alt+A, or hold Alt+A for a brief annotation. Thickness is remembered per tool.

ToolKeyBehavior
ArrowAAn arrow from the start to the end point.
RectangleRA rectangular outline.
EllipseCAn elliptical outline.
LineSA straight segment.
PencilWA freehand opaque stroke with smoothing.
HighlighterFA translucent stroke with a screen-aligned rectangular nib.
EraserEDeletes complete annotations, not individual pixels; pasted images are left untouched.
TextTAdds text in the selected color and size.
MoveQSelects, moves, and edits objects.
StepDNumbered oval or rectangular markers.

Pencil and Highlighter

Off, Balanced, and Strong smoothing reduce edge jitter. They do not recognize shapes or simulate pressure. The Highlighter has a fixed rectangular nib: a horizontal stroke is broad, a vertical stroke is narrow, and a diagonal stroke has an intermediate visible width. Briefly holding the endpoint straightens the current stroke.

Precise geometry with Shift

Hold Shift while drawing: Rectangle becomes a square, Ellipse becomes a true circle, Step Rectangle becomes a square marker, and Line snaps in 45-degree increments. The same constraints apply while resizing existing shapes; an Arrow endpoint can also snap to 45-degree directions while being edited. A pasted image always resizes with its original aspect ratio preserved.

Eraser

A click removes the top erasable annotation under the pointer; dragging removes intersected annotations as one undoable action. Pasted images are skipped, so they can serve as a background and are removed deliberately with Move and Backspace/Delete. The cursor circle indicates the hit radius and can change color over a target. It is a live tool indicator and is never added to the drawing.

Text, images, and editing

Paste text or an image from the clipboard with Ctrl+V. Enter completes text input and Shift+Enter inserts a new line. Move can box-select one or several objects and drag them. Double-click text to edit it or a supported shape to reveal editing handles. Ctrl+wheel changes the thickness of a selected shape or the size of selected text. Backspace deletes the selection. Step numbering has an independent sequence for each color slot.

History and clearing

Ctrl+Z undoes and Ctrl+Shift+Z redoes. Delete or Shift+E clears annotations. Esc first cancels active editing/selection and then exits Draw, so a sub-mode can require a second press.

Fading Annotations

Ctrl+Alt+D enables temporary annotations. Settings control the visible and fade durations. Use this for brief explanations that should disappear without manual clearing.

Push-to-annotate

Press the complete assigned Push-to-annotate shortcut to enter. Draw then stays active while any key from that shortcut remains held. Keys already down at entry do not execute local commands: with the default Alt+A, Arrow is not selected automatically. Release A while keeping Alt held, then press W, F, 1–5, or A again. After every shortcut key is released, the mode exits once the active stroke has completed. The rule works with any assigned keyboard combination.

Useful combination. Enable Fading Annotations once, press Alt+A, release A while keeping Alt held, select a tool, and draw. Releasing Alt returns control to the source application while the explanation fades after the configured time.

6. Masks and selected regions

Region Mask

Press Ctrl+Alt+R and draw a rectangle. Create multiple masks, drag them, and resize them by their handles. Right-click a mask to change its style or delete it; Delete and Backspace also remove the selected mask. Styles include Solid, Stripes, a HIDE label, and stripes with the label. Choosing a new default style affects future masks; existing masks retain their own style.

Clear all masks from the toolbar or tray. No clear shortcut is assigned by default, but you can configure one.

How it works. A mask visually covers the selected region without changing the source window. When using a third-party recorder, it is useful to confirm beforehand whether its selected capture method includes overlays.

7. Timers

Ctrl+Alt+T creates a timer. Multiple timers are supported and can remain over normal windows and boards.

ModePurpose
CountdownCounts a chosen duration down to zero.
StopwatchCounts elapsed time upward.
ClockShows current local time.
Until timeCounts down to a selected time of day.

The context menu provides Start/Pause, Reset (Countdown and Stopwatch), mode selection, label and time editing, and Close. Styling includes Light, Dark, or Auto theme, 12/24-hour format, progress and label visibility, and finish blinking. Scale and opacity are adjustable. Maximum entered duration is 99:59:59, and up to ten recent labels are retained.

Mouse and keyboard control

Drag the timer body to move it. Double-click the label to edit it, or double-click the time to change it in a compatible mode. When a timer is selected, the following commands are available:

8. Screenshots and boards

Screenshot

Ctrl+Alt+E captures the current monitor. Ctrl+Alt+Shift+E selects a region and displays its exact size. After drawing it, move the frame or resize it by its handles; arrow keys move it by 1 pixel and Shift+arrow by 10. Enter captures and Delete discards the pending region. The PNG is copied to the clipboard and saved automatically under Pictures\FocusTool. The toolbar and cursor-following magnifier are hidden during capture.

Screen Board

Ctrl+Alt+B creates a board from the current screen. FocusTool forms the base image and then restores visual regions above the board; a mask briefly disappearing during the transition is expected. Draw on the board and save the result as a screenshot.

Black Board and White Board

Ctrl+Alt+Shift+B opens a black board, and Ctrl+Alt+W opens a white board. These are clean drawing backgrounds; timers may remain above them. Press Esc to close a board.

Frame composition. FocusTool's own captures include the annotations and visual regions intended for presentation. Third-party software may compose overlays differently depending on its selected capture type.

9. Capture Stage

Capture Stage presents a selected window in a separate borderless view fitted to the display work area, excluding the taskbar. Selection uses the Windows system picker. Multiple Stages can be opened and closed individually or together.

  • Stage is a presentation view; clicks are not forwarded to the source window.
  • It selects an application window, not an individual browser tab.
  • FocusTool elements are composed over the view when their positions can be mapped to the selected window.
  • If the source is closed or lost, Stage reports that it is unavailable; select it again.
  • When minimized, source behavior depends on Windows and the app: the last available frame may remain, or the source may become unavailable.
Compatibility. Some protected or hardware-accelerated windows do not expose an image to system capture. A source application may also draw its own cursor, producing a second pointer next to FocusTool effects.

10. Toolbar and system tray

Open the toolbar with Ctrl+Alt+G. It collects mode toggles, drawing tools, colors, thickness, clear commands, and capture actions. Tooltips show a command name and its current shortcut.

The notification-area icon provides settings, region creation/clearing, timers, lenses, Capture Stage, and group close commands without opening the toolbar. Exit completely with Ctrl+Alt+Shift+Q or tray → Exit. Closing one overlay does not necessarily quit FocusTool.

14. Files, settings, and privacy

  • Captures are saved under Pictures\FocusTool and copied to the clipboard.
  • Settings are stored locally for the current Windows user.
  • Current drawings, timers, lenses, masks, and Stages are session state, not a saved project after the app exits.
  • FocusTool is not a source-document editor and never writes annotations back to the presented application.

Before handing the computer to another person, inspect the clipboard, the capture folder, and remaining overlays. FocusTool does not replace access-control policy or metadata-removal tools.

15. Compatibility and current capabilities

  • Windows 10/11 and the runtime required by the chosen build are supported. Results depend on the Windows version, video driver, DPI scaling, and source application.
  • Shape recognition, pen-pressure support, and a pixel eraser are not currently provided.
  • Shortcut profiles, timer sequences, timer sound, a keystroke overlay, and a webcam feature are outside the current feature set.
  • Capture Stage displays a window but does not control it or select individual browser tabs.
  • Protected media and exclusive full-screen output may not be available to system capture.
  • The contents of a third-party recording depend on the capture method selected in that recorder.
  • Masks visually cover an area without changing the source application's data.

17. Practical workflows

Temporary explanation without manual clearing

  1. Enable Fading Annotations once.
  2. Hold Alt+A and select a tool with its letter key.
  3. Draw and release the shortcut: control returns to the application and the mark disappears automatically.

Focus and magnify a small element

  1. Enable Spotlight.
  2. Enable Magnifier—the area around the enlarged content remains dimmed.
  3. Set zoom with Ctrl+wheel and radius with Ctrl+Shift+wheel.

Electronic board for a lesson

  1. Open White Board for a conventional diagram or Black Board for high contrast.
  2. Use Shift for circles, squares, and 45-degree lines, and Step for sequence.
  3. Add text or paste a prepared image with Ctrl+V.
  4. Save the result with the built-in Screenshot.

Review a static interface

  1. Open Screen Board to freeze the current screen.
  2. Add arrows, frames, text, and numbered steps without the source changing underneath.
  3. Save the annotated frame, or clear it and repeat the explanation.

Step-by-step instruction with branches

  1. Select Step Oval or Step Rect.
  2. Use different color slots for independent numbering sequences.
  3. Connect steps with arrows and add short text notes.

Compare details or states

  1. Create several Pinned Lenses for the relevant regions.
  2. Set each lens zoom with Ctrl+wheel.
  3. Freeze one lens and compare its frame with the changing content in another.

Present a single application

  1. Start Capture Stage and choose a window in the Windows system picker.
  2. Add a pointer, annotations, or a timer over the presentation view.
  3. Switch to the source window to interact with it; Stage remains the presentation view.

Capture a precise region

  1. Open Region Screenshot and draw an approximate region.
  2. Refine the frame with the mouse, arrow keys by 1 pixel, or Shift+arrows by 10.
  3. Check the displayed dimensions and press Enter.

Show several important areas

  1. Create several Region Spotlights around related elements.
  2. Move and fine-tune selected regions with the arrow keys.
  3. Delete completed regions individually or clear all from the toolbar.

Reveal material progressively

  1. Cover not-yet-discussed fragments with several Region Masks.
  2. Move or delete selected masks as the explanation progresses.
  3. For recording, confirm the recorder's selected capture type beforehand.

Keep a presentation on schedule

  1. Create a Countdown for the overall duration.
  2. Add a Stopwatch for an exercise or question segment.
  3. Control the selected timer with Space, R, and arrow keys without opening a menu.

Make mouse actions easy to follow

  1. Enable Cursor Highlight so the pointer position is consistently visible.
  2. Add Click Pulse for feedback on clicks.
  3. Hold Laser temporarily when you need to point out the next element.
