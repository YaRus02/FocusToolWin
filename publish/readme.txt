FocusTool
=========

FocusTool - минималистичная Windows-утилита для демонстраций, уроков,
созвонов, записи видео, разборов интерфейсов и работы с экраном.

Приложение добавляет поверх рабочего стола:

- лазерный указатель;
- стабильную подсветку курсора;
- click pulse для отображения кликов;
- spotlight вокруг курсора;
- прямоугольный region spotlight;
- экранную лупу;
- live pinned lens;
- region masks;
- инструменты рисования и текста;
- нумерованные step-маркеры;
- исчезающие аннотации;
- вставку изображений и текста из буфера обмена;
- overlay timer;
- screen board, black board и white board;
- скриншоты монитора и выбранной области;
- compact overlay toolbar и tray menu.

Главная идея FocusTool - минимум визуального шума и быстрый доступ через
горячие клавиши, tray menu и компактную панель поверх экрана.

Документация актуальна для версии 2.2.0.


1. Какой файл запускать
-----------------------

В публикации обычно есть две сборки:

self-contained\FocusTool.exe
  Автономная версия. Не требует установленного .NET Runtime.
  Рекомендуется для обычного использования и распространения.

framework-dependent\FocusTool.exe
  Облегченная версия. Требует установленный .NET Desktop Runtime
  подходящей версии. Удобна, если runtime уже установлен.

Для обычного использования запускайте:

  self-contained\FocusTool.exe

После запуска FocusTool появляется в системном трее. Если значок не виден,
проверьте скрытые значки трея Windows.

Исполняемый файл пока не подписан цифровой подписью. Windows SmartScreen
может показать предупреждение при первом запуске нового неподписанного
приложения.


2. Где хранятся настройки и файлы
---------------------------------

Настройки:

  %APPDATA%\FocusTool\settings.json

Лог ошибок:

  %APPDATA%\FocusTool\log.txt

Скриншоты и сохраненные screen board:

  Pictures\FocusTool

Если папка Pictures недоступна, приложение использует Documents.


3. Общая логика работы
----------------------

FocusTool создает overlay-окна поверх мониторов. В обычном состоянии они
пропускают клики и не мешают работе с другими приложениями.

Когда включается Draw mode, выбор области, редактирование масок, region
spotlight или screen board, overlay начинает принимать ввод мыши и клавиатуры.

Приложение не показывает основное рабочее окно. Управление идет через:

- системный трей;
- overlay toolbar;
- глобальные хоткеи;
- контекстные меню timer, pinned lens и region mask;
- окно Settings.


4. Основные горячие клавиши по умолчанию
----------------------------------------

Глобальные режимы:

  Ctrl+Alt+L        - переключить режим лазера Always / Hold
  Ctrl+Alt+D        - включить или выключить Draw mode
  Alt+A             - Push-to-annotate: временный Draw mode по удержанию
  Ctrl+Alt+U        - включить или выключить Cursor Highlight
  Ctrl+Alt+S        - включить или выключить Spotlight
  Ctrl+Alt+M        - включить или выключить Magnifier
  Ctrl+Alt+P        - выбрать область для Live Pinned Lens
  Ctrl+Alt+H        - выбрать область для Region Mask
  Ctrl+Alt+Shift+H  - очистить все Region Masks
  Ctrl+Alt+Shift+S  - выбрать область для Region Spotlight
  Ctrl+Alt+Shift+X  - очистить все Region Spotlights
  Ctrl+Alt+F        - включить или выключить Fading annotations
  Ctrl+Alt+N        - создать новый Overlay Timer
  Ctrl+Alt+T        - показать или скрыть Overlay Toolbar
  Ctrl+Alt+C        - сделать скриншот текущего монитора
  Ctrl+Alt+Shift+C  - сделать скриншот выбранной области
  Ctrl+Alt+G        - включить или выключить Screen Board
  Ctrl+Alt+B        - включить или выключить Black Board
  Ctrl+Alt+W        - включить или выключить White Board
  Ctrl+Alt+Q        - выйти из FocusTool

Удержание в Hold-режимах:

  Alt+Z             - удержание лазера в режиме Hold
  Alt+X             - удержание Cursor Highlight в режиме Hold

Draw tools:

  A                 - Arrow
  R                 - Rectangle
  C                 - Oval / Circle
  L                 - Line
  P                 - Pen
  H                 - Highlighter
  T                 - Text
  M                 - Move
  N                 - Step marker

Цвета и размер:

  1                 - color slot 1
  2                 - color slot 2
  3                 - color slot 3
  4                 - color slot 4
  5                 - color slot 5
  [                 - уменьшить толщину линии
  ]                 - увеличить толщину линии

Команды аннотаций:

  Ctrl+Z            - Undo
  Ctrl+Y            - Redo
  Backspace         - удалить выбранный объект
  Delete            - очистить аннотации
  E                 - альтернативная очистка аннотаций
  Esc               - выйти из визуального режима
  Ctrl+V            - вставить изображение или текст из буфера

Любую комбинацию можно изменить или отключить в Settings -> Shortcuts.
Чтобы отключить хоткей, оставьте поле пустым или укажите None.


5. Важное про keyboard ghosting
-------------------------------

Некоторые клавиатуры физически не передают в систему отдельные комбинации
одновременно зажатых клавиш. Это называется keyboard ghosting или ограничение
key rollover.

Например, сочетание Ctrl+Space+3 или Ctrl+Space+E на одной клавиатуре может
не доходить до приложения, хотя Ctrl+Space+A или другие клавиши работают.
В таком случае FocusTool не может обработать нажатие, потому что Windows его
фактически не получает.

По этой причине Push-to-annotate по умолчанию использует Alt+A, а удержание
лазера и Cursor Highlight - Alt+Z и Alt+X.

Если конкретное сочетание не работает на вашей клавиатуре, назначьте другое
в Settings -> Shortcuts.


6. Laser Pointer
----------------

Laser Pointer рисует цветную точку и плавный хвост поверх экрана.

Поддерживается:

- режим Always;
- режим Hold;
- настраиваемые цветовые слоты;
- размер точки;
- длина trail;
- fade после отпускания;
- glow.

Ctrl+Alt+L переключает режим активации Always / Hold.
В Hold-режиме по умолчанию используется Alt+Z.


7. Cursor Highlight и Click Pulse
---------------------------------

Cursor Highlight - стабильное кольцо вокруг курсора. Оно не использует
логику лазерного хвоста, не "набирает" цвет при движении и не мерцает.

Подходит для:

- записи экрана;
- вебинаров;
- демонстрации интерфейсов;
- спокойной постоянной подсветки положения курсора.

Click Pulse показывает короткую анимацию вокруг курсора при клике мыши.
Он включается отдельно и может использоваться как вместе с Cursor Highlight,
так и без постоянного кольца. Он нужен, чтобы зритель видел факт нажатия
без дополнительных пояснений.

В настройках доступны:

- цветовые слоты Cursor Highlight;
- режим Always / Hold;
- hold shortcut;
- размер;
- толщина кольца;
- включение / отключение Click Pulse.

В Hold-режиме по умолчанию используется Alt+X.


8. Spotlight и Region Spotlight
-------------------------------

Spotlight затемняет экран вокруг курсора и оставляет видимой круглую область.

  Ctrl+Alt+S

Region Spotlight - прямоугольная версия spotlight.

  Ctrl+Alt+Shift+S

Поведение Region Spotlight:

- пользователь выделяет прямоугольную область;
- внутри области экран остается видимым;
- все вокруг затемняется;
- область можно двигать;
- размер можно менять за углы;
- стрелки сдвигают выбранный регион;
- Backspace удаляет выбранный регион;
- Enter и Esc выходят из режима редактирования без удаления.

Это удобно для выделения таблицы, панели интерфейса, блока кода или фрагмента
настройки, когда круговой spotlight недостаточно точен.


9. Magnifier
------------

Magnifier увеличивает область вокруг курсора.

  Ctrl+Alt+M

Настройки:

- radius;
- zoom.

Лупа использует Windows Magnification API. Region masks остаются видимыми
в увеличенной области, чтобы скрытый фрагмент нельзя было раскрыть через zoom.
Pointer visuals вроде Laser / Cursor Highlight подавляются при активной
лупе, чтобы не создавать артефакты внутри увеличенного изображения.


10. Live Pinned Lens
--------------------

Live Pinned Lens - живой увеличенный фрагмент выбранной области экрана.

Запуск:

  Ctrl+Alt+P

После запуска выделите прямоугольник на экране. FocusTool создаст плавающее
окно с увеличенной live-копией выбранной области.

Поддерживается:

- несколько pinned lens одновременно;
- перетаскивание окна;
- тонкая обводка без стандартной системной рамки;
- контекстное меню по правой кнопке;
- Freeze / Unfreeze;
- Zoom in / Zoom out;
- Ctrl + колесо мыши для zoom конкретной линзы;
- Close / Close all;
- удаление выбранной линзы через Backspace / Delete.

Аннотации рисуются поверх pinned lens. Region masks остаются видимыми внутри
увеличенного изображения.


11. Region Mask
---------------

Region Mask скрывает выбранные прямоугольные области экрана.

Запуск:

  Ctrl+Alt+H

Очистить все маски:

  Ctrl+Alt+Shift+H

Поведение:

- можно создать несколько масок подряд;
- маска является верхним privacy-layer;
- маски видны в screenshots, screen board, magnifier и pinned lens;
- выбранную маску можно двигать;
- размер можно менять за углы;
- Backspace / Delete удаляет выбранную маску;
- Ctrl + колесо мыши меняет opacity выбранной маски;
- цвет выбранной маски можно менять live;
- контекстное меню позволяет удалить маску и выбрать стиль.

Стили:

- solid fill;
- diagonal stripes;
- HIDE label;
- stripes + HIDE label.

Новые маски используют текущий цвет, opacity и последний выбранный стиль.
Уже созданные маски сохраняют собственные параметры.


12. Draw mode и аннотации
-------------------------

Draw mode:

  Ctrl+Alt+D

Инструменты:

- Pen;
- Highlighter;
- Arrow;
- Line;
- Rectangle;
- Oval;
- Text;
- Move;
- Step marker.

Step marker:

- oval step ставится кликом;
- rect step рисуется выделением прямоугольника;
- номер вычисляется автоматически;
- нумерация идет отдельно по цветовому слоту;
- Clear сбрасывает последовательность.

Редактирование:

- двойной клик по объекту включает edit mode;
- вне edit mode обычные инструменты создают новые объекты;
- выбранные объекты можно удалять Backspace;
- цвет, толщина и размер текста могут применяться live;
- Text редактируется на месте;
- Enter завершает ввод текста;
- Esc закрывает или отменяет ввод;
- Shift+Enter добавляет новую строку.

Paste:

  Ctrl+V

Если в буфере есть изображение или текст, FocusTool вставит его как объект
аннотации. Неподдерживаемое содержимое буфера игнорируется.


13. Fading annotations
----------------------

Fading annotations включаются глобально.

  Ctrl+Alt+F

Когда режим включен, новые аннотации сначала видны полностью, затем плавно
исчезают. Уже созданные аннотации не меняют свое поведение при переключении
режима.

Параметры visible time и fade time доступны в toolbar и настройках Draw.


14. Overlay Timer
-----------------

Создать новый таймер:

  Ctrl+Alt+N

Поддерживается:

- несколько таймеров одновременно;
- Countdown;
- Stopwatch;
- Clock;
- Until time;
- пользовательская подпись;
- история введенных подписей;
- двойной клик по подписи для редактирования;
- двойной клик по времени для ввода времени в применимых режимах;
- 12/24-часовой формат;
- progress bar;
- blink on finish;
- масштаб;
- прозрачность;
- dark / light / auto theme;
- контекстное меню управления.

Звука нет намеренно. Таймер рассчитан на спокойное визуальное отображение
поверх текущей работы, а не на полноэкранный режим.


15. Screen Board, Black Board, White Board
------------------------------------------

Screen Board:

  Ctrl+Alt+G

Screen Board делает снимок текущего монитора и превращает его в фон для
аннотаций. При выходе итог автоматически сохраняется в Pictures\FocusTool
и копируется в буфер обмена.

Black Board:

  Ctrl+Alt+B

White Board:

  Ctrl+Alt+W

Black/White Board дают чистый фон для объяснений. Они не делают снимок экрана
при входе, в отличие от Screen Board.


16. Скриншоты
-------------

Скриншот текущего монитора:

  Ctrl+Alt+C

Скриншот выбранной области:

  Ctrl+Alt+Shift+C

При скриншоте:

- изображение сохраняется в Pictures\FocusTool;
- изображение копируется в буфер обмена;
- toolbar временно скрывается;
- аннотации и region masks попадают в результат.

Region Screenshot позволяет выделить область, уточнить рамку перед сохранением
и видеть размер области.

Имена файлов имеют формат:

  FocusTool_yyyy-MM-dd_HH-mm-ss-fff.png

Screen board сохраняется с префиксом:

  FocusTool_Board_yyyy-MM-dd_HH-mm-ss-fff.png


17. Overlay Toolbar
-------------------

Overlay Toolbar - компактная панель управления поверх экрана.

Показать / скрыть:

  Ctrl+Alt+T

Панель содержит группы:

- Pointer: Laser и Cursor Highlight;
- Spot;
- Zoom и Pin;
- Draw;
- Mask;
- Board;
- Capture;
- Timer;
- дополнительные действия через "...".

У основных групп есть контекстные ряды с быстрыми настройками. Панель можно
свернуть в маленький grip с надписью FT и развернуть обратно кликом.


18. Цветовые слоты
------------------

FocusTool использует настраиваемые цветовые слоты:

- Laser: Color 1..5;
- Cursor Highlight: Color 1..5;
- Annotation: Color 1..5;
- Region Mask: Color 1..5.

Цвета задаются в HEX:

  #FFFF2020
  #80FFFFFF
  #BFFFD400

Поддерживается alpha-канал в формате #AARRGGBB.
Например, #80FFFFFF - белый примерно с 50% прозрачности.


19. Tray menu и Settings
------------------------

FocusTool работает из системного трея. В tray menu доступны основные режимы,
цвета, инструменты, undo/redo/clear, settings и exit.

Settings разделены по вкладкам:

- Pointer;
- Focus;
- Draw;
- Mask;
- Shortcuts.

В Shortcuts можно изменить или отключить глобальные хоткеи, хоткеи инструментов,
цветов и команд.

Формат хоткеев:

  A
  Alt+A
  Ctrl+Alt+D
  Delete
  Esc
  Backspace
  [
  ]

Для отключения используйте пустое поле или None.


20. Запись и стриминг
---------------------

FocusTool рисует как desktop overlay. Для OBS и похожих программ используйте
Display Capture / Monitor Capture, если нужно записать лазер, аннотации,
spotlight, magnifier, pinned lens, region masks, timer и toolbar.

Window Capture обычно захватывает только окно конкретного приложения и может
не включать overlay-окна. Это ограничение композиции окон Windows, а не ошибка
FocusTool.


21. Ограничения и примечания
----------------------------

- FocusTool рассчитан на Windows 10 / 11.
- Для обычного распространения лучше использовать self-contained сборку.
- Неподписанный exe может вызывать предупреждение Windows SmartScreen.
- Если глобальный хоткей занят другой программой, его нужно переназначить.
- Некоторые сочетания клавиш могут не работать из-за keyboard ghosting.
- В Draw mode overlay перехватывает мышь. Для обычной работы выйдите через Esc.
- Screen Board сохраняет итог при выходе. Black Board и White Board не делают
  автосохранение при выходе.
- Производительность зависит от числа мониторов, DPI, количества pinned lens,
  таймеров, масок и активных анимаций.
- Timer не использует звук.


22. Быстрый старт
-----------------

1. Запустите self-contained\FocusTool.exe.
2. Нажмите Ctrl+Alt+T, чтобы открыть overlay toolbar.
3. Нажмите Ctrl+Alt+D, чтобы включить Draw mode.
4. Выберите Pen / Arrow / Rect / Text / Step через toolbar или хоткеи.
5. Нажмите Esc, чтобы выйти из Draw mode.
6. Нажмите Alt+A и удерживайте, чтобы быстро сделать временную пометку.
7. Нажмите Ctrl+Alt+N, чтобы создать overlay timer.
8. Нажмите Ctrl+Alt+P и выделите область для live pinned lens.
9. Нажмите Ctrl+Alt+H и выделите область для region mask.
10. Нажмите Ctrl+Alt+Shift+C, чтобы снять скриншот выбранной области.
11. Нажмите Ctrl+Alt+G, чтобы создать screen board.


23. Удаление
------------

FocusTool не требует установки.

Чтобы удалить приложение:

1. Закройте FocusTool через tray menu или Ctrl+Alt+Q.
2. Удалите папку с exe.
3. При необходимости удалите настройки:

   %APPDATA%\FocusTool

4. При необходимости удалите сохраненные скриншоты:

   Pictures\FocusTool
