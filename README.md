# FocusTool

![Platform](https://img.shields.io/badge/platform-Windows%2010%2F11-0078D6)
![.NET](https://img.shields.io/badge/.NET-10-512BD4)
![License](https://img.shields.io/badge/license-MIT-green)

**FocusTool** — это лёгкая программа для **рисования и аннотаций поверх экрана** в Windows: **виртуальная лазерная указка** с хвостом, **экранная лупа**, **подсветка области (spotlight)**, **скриншоты с пометками** и **доска для рисования поверх любого приложения**. Удобный инструмент для **презентаций, вебинаров, онлайн-уроков, стримов и записи обучающих видео**.

> _Ключевые слова: рисование поверх экрана, аннотации на экране, лазерная указка для презентаций, экранная лупа, увеличение части экрана, прожектор / спотлайт, выделение области экрана, скриншот с рисованием, доска для презентаций, оверлей для стримов, инструмент преподавателя._

> **О разработке.** Приложение создано с помощью ИИ. При этом весь функционал **вручную протестирован во всех режимах работы** — лазер, аннотации, прожектор, лупа, доски, скриншоты, горячие клавиши и многомониторные сценарии.

## Применимость

- **Презентации и доклады** (PowerPoint, Google Slides, PDF) — указка, выделение нужного места, подписи и стрелки прямо поверх слайдов.
- **Онлайн-уроки, вебинары, репетиторство** — рисование поверх любого приложения, чтобы наглядно объяснять.
- **Стримы и запись обучающих видео** — лазер, спотлайт и лупа делают акценты заметными.
- **Демонстрация кода и код-ревью** — подсветить строку, обвести блок, увеличить мелкий шрифт.
- **Техподдержка и удалённая помощь** — наглядно показать, куда нажимать.
- **Быстрые скриншоты с пометками** — снять монитор, сохранить и сразу скопировать в буфер обмена.

## Возможности

- **Лазерная указка** с плавным «кометным» хвостом: настраиваемый цвет, размер, свечение, длина и затухание следа. Два режима — **«всегда включена»** или **«по удержанию»** клавиши/кнопки мыши.
- **Аннотации поверх экрана**: карандаш, маркер, стрелка, линия, прямоугольник, эллипс/круг, текст, а также выделение и перемещение нарисованного. Отмена/повтор, очистка.
- **Spotlight (прожектор)** — затемняет экран, оставляя освещённым круг вокруг курсора.
- **Экранная лупа** — увеличивает область под курсором (на базе Windows Magnification API), в том числе для динамического содержимого.
- **Screen board (стоп-кадр)** — «замораживает» текущий экран как фон-доску, поверх которой можно рисовать.
- **Чёрная и белая доска** — чистый фон для рисования.
- **Скриншот текущего монитора** — сохраняется в `Изображения\FocusTool` и копируется в буфер обмена.
- **Плавающая панель инструментов** — все инструменты под рукой; перетаскивается, сворачивается в компактный грип и запоминает своё положение.
- **Значок в системном трее** с полным меню и настройками.
- **Глобальные горячие клавиши** — все переназначаются в настройках.
- **Многомониторность и поддержка масштабирования** (Per-Monitor DPI), корректная работа на дисплеях 125 % / 150 %.

## Горячие клавиши (по умолчанию)

### Глобальные

| Действие | Клавиши |
|---|---|
| Режим аннотаций (вкл/выкл) | `Ctrl+Alt+D` |
| Лазер: режим «всегда / по удержанию» | `Ctrl+Alt+L` |
| Прожектор (spotlight) | `Ctrl+Alt+S` |
| Лупа | `Ctrl+Alt+M` |
| Панель инструментов | `Ctrl+Alt+T` |
| Скриншот текущего монитора | `Ctrl+Alt+C` |
| Screen board (стоп-кадр) | `Ctrl+Alt+G` |
| Чёрная доска | `Ctrl+Alt+B` |
| Белая доска | `Ctrl+Alt+W` |
| Выход из приложения | `Ctrl+Alt+Q` |
| Удержание лазера (в режиме «по удержанию») | боковая кнопка мыши `XButton2` |

### В режиме аннотаций

| Действие | Клавиши |
|---|---|
| Инструменты | `A` стрелка · `R` прямоугольник · `C` круг · `L` линия · `P` карандаш · `H` маркер · `T` текст · `M` перемещение |
| Цвета | `1`–`5` |
| Толщина линии | `[` и `]` |
| Отмена / повтор | `Ctrl+Z` / `Ctrl+Y` |
| Удалить выделенное | `Backspace` |
| Очистить всё | `Delete` или `E` |
| Выйти из режима | `Esc` |

Любую комбинацию можно изменить или отключить в окне настроек (правый клик по значку в трее → **Settings…**).

## Установка и запуск

**Требования:** Windows 10 / 11.

- **Готовая сборка:** распакуйте релиз и запустите `FocusTool.exe`. Приложение свернётся в системный трей. Для self-contained-сборки отдельная установка .NET не нужна; для обычной — потребуется **.NET 10 Desktop Runtime**.
- Приложение запускается в единственном экземпляре; управление — через значок в трее и горячие клавиши.

## Запись и стриминг (OBS и подобные)

Аннотации, лазер, спотлайт и лупа рисуются отдельным слоем **поверх всего экрана**, а не внутри конкретного приложения. Поэтому при режиме **«Захват окна» (Window Capture)** другой программы они **не попадут в кадр** — это особенность того, как устроена композиция окон в Windows, а не ограничение FocusTool.

Для записи и трансляций используйте **«Захват экрана / монитора» (Display Capture)** — он захватывает весь рабочий стол вместе с оверлеем. Чтобы в кадре была только нужная область, наложите фильтр обрезки (Crop/Pad).

## Сборка из исходников

Требуется **.NET 10 SDK**.

```bash
# Сборка
dotnet build -c Release

# Запуск
dotnet run -c Release

# Публикация одиночного .exe (self-contained, без установленного .NET)
dotnet publish -c Release -r win-x64 --self-contained true ^
  -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true
```
