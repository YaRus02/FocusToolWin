# FocusTool

![Platform](https://img.shields.io/badge/platform-Windows%2010%2F11-0078D6)
![.NET](https://img.shields.io/badge/.NET-10-512BD4)
![License](https://img.shields.io/badge/license-MIT-green)

**FocusTool** — минималистичная Windows-утилита для презентаций, занятий, разборов, созвонов, записи видео и работы с экраном. Приложение добавляет поверх рабочего стола лазерную указку, аннотации, spotlight, лупу, живые закреплённые увеличенные области, маски областей, экранные доски, скриншоты и компактную overlay-панель.

Основная идея: минимум визуального шума, максимум управления через горячие клавиши и быстрые контекстные элементы.

## Возможности

- **Лазерная указка** с режимами Always и Hold, настраиваемым размером точки, следом, затуханием, свечением и цветовыми слотами.
- **Аннотации поверх экрана**: Pen, Mark, Arrow, Line, Rect, Oval, Text, Move.
- **Исчезающие аннотации**: новые аннотации могут жить заданное время и затем плавно исчезать.
- **Spotlight**: затемнение экрана с видимой областью вокруг курсора.
- **Лупа** на базе Windows Magnification API.
- **Live Pinned Lens**: выбранная область экрана показывается увеличенной в отдельном живом плавающем окне.
- **Region Mask**: скрытие выбранных прямоугольных областей экрана.
- **Screen Board**: стоп-кадр текущего монитора как доска для рисования.
- **Black Board / White Board**: чистые фоны для быстрых объяснений.
- **Скриншоты текущего монитора** с автоматическим сохранением и копированием в буфер обмена.
- **Overlay toolbar**: компактная панель с контекстными рядами для Laser, Draw, Spot, Zoom, Pin, Mask, Board и служебных действий.
- **Tray menu** с основными режимами, статусом и настройками.
- **Глобальные горячие клавиши**, переназначаемые в Settings.
- **Многомониторность и Per-Monitor DPI**.

## Overlay Toolbar

Показать или скрыть панель:

```text
Ctrl+Alt+T
```

Основной ряд панели:

- `Laser` — переключение режима лазера и настройки лазера.
- `Draw` — режим аннотаций, инструменты, цвета, размер линии, размер текста, Fade, Undo, Redo, Clear.
- `Spot` — spotlight и его настройки.
- `Zoom` — лупа и её настройки.
- `Pin` — выбор области для Live Pinned Lens и настройки новых линз.
- `Mask` — выбор области Region Mask, цвет и прозрачность новых масок.
- `Board` — Screen Board, Black Board, White Board.
- `Shot` — скриншот текущего монитора.
- `...` — Hide, Close, Settings.

Панель можно свернуть в маленький grip `FT`. Grip можно перетаскивать; обычный клик раскрывает панель обратно.

## Цветовые слоты

FocusTool использует настраиваемые цветовые слоты вместо жёстких имён цветов.

- Laser: `Color 1`–`Color 5`.
- Annotation: `Color 1`–`Color 5`.
- Region Mask: `Color 1`–`Color 5`.

Цвета редактируются только в Settings через HEX-значения:

```text
#FFFF2020
#FF2080FF
#FFFFFFFF
```

Toolbar и tray используют те же самые слоты. Если в старом файле настроек был пользовательский текущий цвет, которого нет в слотах, FocusTool сохраняет его в `Color 5`, чтобы он отображался во всех слоях управления.

## Live Pinned Lens

Live Pinned Lens — это живой увеличенный выносной фрагмент экрана.

Сценарий:

1. Запустить выбор через `Ctrl+Alt+P`, toolbar или tray.
2. Выделить прямоугольную область на экране.
3. Перетащить увеличенное окно в удобное место.

Поддерживается:

- несколько pinned lens окон одновременно;
- borderless-окно с тонкой обводкой;
- перетаскивание мышью;
- контекстное меню по ПКМ;
- Freeze / Resume;
- Zoom in / Zoom out;
- `Ctrl + колесо мыши` для изменения zoom конкретной линзы;
- Close и Close all.

Лазер и аннотации отображаются поверх pinned lens. Region masks остаются видимыми в увеличенном изображении, чтобы замаскированную область нельзя было раскрыть лупой или линзой.

## Region Mask

Region Mask скрывает выбранные прямоугольные области экрана.

Запуск:

```text
Ctrl+Alt+H
```

Поведение:

- одно выделение создаёт одну маску и возвращает приложение к обычному режиму;
- в режиме Mask существующие маски можно перемещать за тело;
- размер маски можно менять за угловые маркеры;
- ПКМ по маске открывает удаление;
- удаление одной маски не выходит из режима Mask;
- очистить все маски можно через `Ctrl+Alt+Shift+H` или tray;
- новые маски используют текущий цветовой слот и opacity из настроек.

## Исчезающие аннотации

Исчезающие аннотации — глобальный режим. Когда он включён, новые аннотации становятся временными: сначала отображаются полностью, затем плавно исчезают.

Уже созданные аннотации не меняют своё поведение при переключении режима. Автоматическое исчезновение не оставляет пустых шагов в undo/redo.

Переключение:

```text
Ctrl+Alt+F
```

В toolbar: `Draw -> Fade`. Маленькая кнопка `v` рядом с Fade открывает быстрые настройки времени полной видимости и длительности исчезновения.

## Скриншоты и доски

Скриншот текущего монитора:

```text
Ctrl+Alt+C
```

Изображение сохраняется в `Pictures\FocusTool` и копируется в буфер обмена. Toolbar временно скрывается, чтобы не попасть в кадр.

Screen Board делает снимок текущего монитора и превращает его в доску для рисования. При выходе итоговая картинка автоматически сохраняется и копируется в буфер.

Black Board и White Board дают чистый фон для рисования и не сохраняются автоматически при выходе.

## Где хранятся данные

| Что | Путь |
|---|---|
| Настройки | `%APPDATA%\FocusTool\settings.json` |
| Лог ошибок | `%APPDATA%\FocusTool\log.txt` |
| Скриншоты и доски | `Pictures\FocusTool` |

Если `Pictures` недоступна, используется `Documents`.

## Запись и стриминг

FocusTool рисует как desktop overlay. Для OBS и похожих программ используйте Display Capture / Monitor Capture, если нужно записать лазер, аннотации, spotlight, лупу, маски и toolbar. Window Capture обычно захватывает только окно приложения и может не включать overlay.

## Установка

FocusTool портативный. Распакуйте релиз и запустите `FocusTool.exe`.

Для большинства пользователей:

```text
self-contained\FocusTool.exe
```

Self-contained сборка не требует установленного .NET Runtime. Framework-dependent сборка меньше, но требует подходящий .NET Desktop Runtime.

Неподписанный exe может вызвать предупреждение Windows SmartScreen. Это нормально для новых неподписанных утилит.

## Сборка из исходников

Требуется .NET 10 SDK.

```powershell
dotnet build -c Release
dotnet run -c Release
```

Пример self-contained публикации:

```powershell
dotnet publish -c Release -r win-x64 --self-contained true `
  -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true
```
