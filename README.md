# FocusTool

FocusTool — компактная Windows-утилита для презентаций, онлайн-занятий, записи видео и разбора интерфейсов. Она добавляет поверх рабочего стола указатели, подсветку, рисование, визуальные маски, таймеры, экранные доски, увеличение и инструменты захвата.

Главный принцип — быстрый доступ к часто используемым действиям с минимумом визуального шума. Режимы включаются глобальными хоткеями, через системный трей или компактную overlay-панель.

Документация актуальна для версии **3.2.0**.

> Примечание к разработке: при создании FocusTool использовалась AI-помощь как инженерный инструмент. Архитектура, интеграция, проверка поведения и финальные изменения выполнялись и контролировались вручную.

## Быстрый старт

1. Распакуйте релиз и запустите `FocusTool.exe`.
2. Откройте панель через `Ctrl+Alt+G` или меню значка в системном трее.
3. Для короткого указания удерживайте `Alt+Z` — Laser или `Alt+X` — Cursor Highlight.
4. Для рисования нажмите `Ctrl+Alt+A`; для временного Draw удерживайте `Alt+A`.
5. Нажмите `Esc`, чтобы отменить текущее редактирование или выйти из интерактивного режима.

Все сочетания можно изменить или отключить в `Settings → Shortcuts`.

## Основные возможности

### Указатель и фокус

- **Laser Pointer** — цветная точка с плавным исчезающим следом, свечением и режимами Always/Hold.
- **Cursor Highlight** — стабильное кольцо вокруг курсора с независимыми цветом, радиусом и толщиной.
- **Click Pulse** — короткая визуальная обратная связь в точке щелчка.
- **Spotlight** — затемнение экрана вне круглой области у курсора.
- **Magnifier** — курсорная лупа; вместе со Spotlight образует увеличенную светлую область на затемнённом экране.
- **Region Spotlight** — одна или несколько прямоугольных областей фокуса, которые можно перемещать, масштабировать и точно сдвигать стрелками.
- **Pinned Lens** — отдельные увеличенные области экрана с масштабированием и Freeze/Unfreeze.

### Рисование и аннотации

- Arrow, Line, Rectangle, Ellipse, Pencil, Highlighter, Eraser, Text, Move и нумерованные Step-маркеры.
- Сглаживание Pencil и Highlighter: Off, Balanced или Strong.
- Highlighter использует прямоугольное перо; короткое удержание конечной точки выпрямляет штрих.
- `Shift` превращает Rectangle в квадрат, Ellipse в круг и фиксирует Line по направлениям с шагом 45°.
- Двойной щелчок открывает редактирование текста или поддерживаемой фигуры.
- `Ctrl+V` вставляет текст или изображение как перемещаемый объект с сохранением пропорций.
- Object Eraser удаляет аннотации целиком и не затрагивает вставленные изображения.
- Undo/Redo, выделение нескольких объектов, раздельная память толщины инструментов и пять цветовых слотов.
- **Fading Annotations** автоматически скрывает временные пояснения.
- **Push-to-annotate** временно включает Draw, пока удерживается `Alt+A`, и не обрывает незавершённый штрих.

Удобная комбинация для живого показа: включить Fading Annotations один раз, затем удерживать `Alt+A`, рисовать и отпускать — управление возвращается исходному приложению, а пояснение исчезает самостоятельно.

### Маски, доски и захват

- **Region Mask** — визуальное перекрытие выбранных областей со стилями Solid, Stripes и `HIDE`. Маски перемещаются, масштабируются и сохраняют собственное оформление.
- **Screen Board** — статичный снимок текущего монитора как фон для дальнейшей разметки.
- **Black Board / White Board** — чистый фон для объяснения или электронной доски.
- **Screenshot** — снимок текущего монитора в PNG, буфер обмена и `Pictures\FocusTool`.
- **Region Screenshot** — настраиваемая рамка с точным размером; стрелки сдвигают её на 1 пиксель, `Shift+стрелка` — на 10.
- **Capture Stage** — отдельное безрамочное представление выбранного окна вместе с оверлеями FocusTool для оконного захвата или демонстрации.

Region Mask предназначен для визуального перекрытия: он не изменяет данные исходного приложения. Состав сторонней записи зависит от выбранного в рекордере метода захвата.

### Таймеры

- Несколько независимых плавающих таймеров.
- Режимы Countdown, Stopwatch, Clock и Until time.
- Подпись, тема, формат 12/24 часа, progress bar, масштаб, прозрачность и мигание по завершении.
- Быстрое управление выбранным таймером: `Space` — старт/пауза, `R` — сброс, `Tab` — следующий режим, стрелки — коррекция времени.
- Текущие таймеры являются состоянием сеанса и не восстанавливаются после перезапуска.

## Практические варианты использования

- **Быстрое пояснение:** Fading Annotations + Push-to-annotate.
- **Показ мелкого интерфейса:** Spotlight + Magnifier.
- **Электронная доска:** White/Black Board, правильные фигуры через `Shift`, Step и вставка материалов.
- **Разбор статичного экрана:** Screen Board, стрелки, рамки, текст и нумерованные шаги.
- **Сравнение состояний:** несколько Pinned Lens, одна из которых заморожена.
- **Показ одного приложения:** Capture Stage с указателем, аннотациями и таймером.
- **Постепенное раскрытие:** несколько Region Mask, удаляемых по ходу объяснения.
- **Демонстрация действий мышью:** Cursor Highlight + Click Pulse + временный Laser.

## Установка

Поддерживаются **Windows 10 и Windows 11**.

Обычно релиз содержит две сборки:

- `self-contained.zip` — автономный вариант без предварительной установки .NET Runtime;
- `framework-dependent.zip` — меньший архив, требующий **.NET 10 Desktop Runtime**.

Для большинства пользователей удобнее `self-contained.zip`: распакуйте архив и запустите `FocusTool.exe`. Приложение появится в системном трее.

Исполняемый файл пока не подписан цифровой подписью, поэтому Windows SmartScreen может показать стандартное предупреждение для нового неподписанного приложения.

## Запись и демонстрация

Для захвата всего рабочего стола в OBS и похожих программах используйте Display/Monitor Capture. Захват отдельного окна обычно не включает сторонние overlay-окна.

Если необходимо показывать только одно приложение вместе с элементами FocusTool, используйте Capture Stage и захватывайте его как обычное окно. Stage является view-only представлением: взаимодействие продолжается в исходном приложении. Источник выбирается системным picker Windows на уровне окон, а не вкладок браузера.

Некоторые защищённые или аппаратно ускоренные поверхности могут не предоставлять изображение системному захвату.

## Совместимость и текущий набор функций

- Глобальный хоткей может конфликтовать с Windows или другим приложением; его можно переназначить или отключить.
- Некоторые комбинации не регистрируются из-за аппаратного keyboard ghosting конкретной клавиатуры.
- Производительность зависит от числа мониторов, DPI, количества активных линз, Stage, таймеров и анимаций.
- Распознавание фигур, чувствительность к нажатию, пиксельный ластик, профили хоткеев, последовательности таймеров и звуковое оповещение сейчас не предусмотрены.

## Сборка из исходников

Требуется **.NET 10 SDK**.

```powershell
dotnet build -c Release
dotnet run -c Release
```

Публикация автономного single-file варианта:

```powershell
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true
```

Verification-проверки:

```powershell
dotnet run --project Verification\FocusTool.Verification.csproj
```
